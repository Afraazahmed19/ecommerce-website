# 🛒 E-Commerce Website

## 📌 Project Overview

This project is a simple **E-Commerce Web Application** developed using **HTML, CSS, JavaScript, PHP, and MySQL**.
The system allows users to browse products, add items to a cart, enter shipping details, and generate an order receipt.

The project demonstrates full-stack web development concepts including frontend UI design, backend logic, and database integration.

---

## 🚀 Features

* User Login System
* Product Listing
* Add to Cart Functionality
* Shipping Information Form
* Order Receipt Generation
* Database Connectivity
* Simple and Responsive UI

## ⚙️ Installation & Setup

1️⃣ Clone the repository

```
git clone https://github.com/yourusername/ecommerce-website.git
```

2️⃣ Move the project to **XAMPP htdocs folder**

```
C:\xampp\htdocs\ecommerce-website
```

3️⃣ Start the following services in **XAMPP**

* Apache
* MySQL

4️⃣ Open **phpMyAdmin**

Create a database:

```
ecommerce_db
```

Import the SQL file if available.

5️⃣ Run the project in browser

```
http://localhost/ecommerce-website
```
